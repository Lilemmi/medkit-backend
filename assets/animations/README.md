# Анимация загрузки приложения

## Как добавить анимацию

1. **Подготовьте файл анимации:**
   - Формат: Lottie JSON (`.json`)
   - Рекомендуемый размер: 512x512 или 1024x1024 пикселей
   - Оптимизируйте анимацию для уменьшения размера файла

2. **Разместите файл:**
   - Скопируйте файл анимации в эту папку (`assets/animations/`)
   - Переименуйте файл в `splash-animation.json`

3. **Альтернативное расположение:**
   - Если файл находится в папке `dev/`, переименуйте его в `splash-animation.json`
   - Или скопируйте в `assets/animations/splash-animation.json`

## Где найти Lottie анимации

- [LottieFiles](https://lottiefiles.com/) - бесплатные и платные анимации
- [IconScout](https://iconscout.com/lottie-animations) - коллекция анимаций
- Создайте свою анимацию в After Effects с плагином Bodymovin

## Требования к анимации

- Формат: JSON (Lottie)
- Размер: рекомендуется до 500KB для быстрой загрузки
- Длительность: 2-3 секунды для оптимального UX
- Цвета: учитывайте цвет фона приложения (#E6F4FE)

## Текущая настройка

Если файл анимации не найден, приложение будет использовать статичный splash screen с логотипом.

## Проверка

После добавления файла перезапустите приложение:
```bash
npm start
```

Анимация автоматически появится при загрузке приложения.
